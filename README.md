# Deep Learning Portfolio

Welcome to my Deep Learning Portfolio! This repository contains an advanced collection of projects demonstrating my expertise in Artificial Neural Networks, Computer Vision, Natural Language Processing, and Generative AI.

## 🚀 Projects Overview

| Project | Domain | Key Algorithms / Techniques | Highlights |
|---------|--------|-----------------------------|------------|
| **[01. Waste Classification CNN](./01_Waste_Classification_CNN/)** | Computer Vision | Convolutional Neural Networks (CNN) | Practical application of Deep Learning to classify waste images |
| **[02. Movie Review Sentiment Analysis](./02_Movie_Review_Sentiment_Analysis/)** | NLP | Sentiment Analysis, Text Processing | Utilizing neural networks to analyze and classify textual sentiment |
| **[03. GAN Implementation](./03_GAN_Implementation/)** | Generative AI | Generative Adversarial Networks | Built and trained a GAN architecture for data generation |
| **[04. LLM Fine Tuning](./04_LLM_Fine_Tuning/)** | NLP / GenAI | Large Language Models (LLMs) | Fine-tuning a pre-trained LLM for specialized downstream tasks |
| **[05. Neural Networks From Scratch](./05_Neural_Networks_From_Scratch/)** | Core AI | MLP, SOM, Perceptron, Adaline | Built fundamental neural network architectures from mathematical scratch |
| **[06. Image Classification (ResNet vs Inception)](./06_Image_Classification_ResNet_Inception/)** | MLOps / CV | ResNet50, InceptionV3, PyTorch | End-to-end classification pipeline comparing models, with serialized `.pth` weights |

## 🛠️ Technologies & Skills
* **Languages:** Python
* **Frameworks/Libraries:** `PyTorch`, `TensorFlow`, `Keras`, `scikit-learn`, `numpy`, `pandas`
* **Core Skills:** Computer Vision, Natural Language Processing, Generative AI (LLMs & GANs), Model Serialization/Deployment.

## ⚙️ How to Run
To run any of the notebooks or scripts locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/WajidAyub/Deep-Learning-Portfolio.git
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Navigate to the project folder and run the relevant scripts or Jupyter Notebooks.
