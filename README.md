# 🧠 Deep Learning Portfolio

Welcome to my **Deep Learning Engineering Portfolio**. This repository is a curated collection of advanced projects demonstrating my expertise in designing, training, and deploying Artificial Neural Networks. 

It covers a wide spectrum of AI domains, including **Computer Vision**, **Natural Language Processing (NLP)**, **Generative AI**, and fundamental neural network mathematics.

## 🚀 Projects Overview

| Project | Domain | Key Technologies & Architectures | Highlights |
|---------|--------|-----------------------------|------------|
| **[01. Waste Classification CNN](./01_Waste_Classification_CNN/)** | Computer Vision | `TensorFlow`, `Keras`, CNNs | Engineered a Convolutional Neural Network from scratch to classify waste images, featuring automated evaluation plotting. |
| **[02. Movie Review Sentiment Analysis](./02_Movie_Review_Sentiment_Analysis/)** | NLP | `TensorFlow`, `NLTK`, LSTMs | Built an end-to-end NLP pipeline for text vectorization and sentiment classification of movie reviews. |
| **[03. GAN Implementation (DCGAN)](./03_GAN_Implementation/)** | Generative AI | `Keras`, `TensorFlow`, GANs | Trained a Deep Convolutional Generative Adversarial Network to synthesize highly realistic MNIST digits from random noise. |
| **[04. LLM Fine-Tuning](./04_LLM_Fine_Tuning/)** | NLP / GenAI | `PyTorch`, `Hugging Face`, LoRA, PEFT | Fine-tuned `distilgpt2` using Low-Rank Adaptation (LoRA) on a custom-curated dataset to adopt a specialized teaching persona. |
| **[05. Neural Networks From Scratch](./05_Neural_Networks_From_Scratch/)** | Core AI Math | `NumPy`, Linear Algebra | Derived and built Perceptron, Adaline, SOM, and an MLP (with full backpropagation) entirely from mathematical scratch. |
| **[06. Transfer Learning (ResNet vs Inception)](./06_Image_Classification_ResNet_Inception/)** | MLOps / CV | `PyTorch`, `Streamlit`, ResNet50, InceptionV3 | Deployed an interactive Streamlit web application comparing the transfer learning efficacy of two state-of-the-art CNNs. |

## 🛠️ Technical Stack
* **Languages:** Python
* **Deep Learning Frameworks:** `PyTorch`, `TensorFlow`, `Keras`
* **Machine Learning Libraries:** `scikit-learn`, `NumPy`, `Pandas`
* **Model Deployment:** `Streamlit`, Hugging Face `peft` / `transformers`
* **Core Competencies:** Computer Vision, Natural Language Processing, Generative AI (LLMs & GANs), Parameter-Efficient Fine-Tuning (PEFT), Model Serialization & Deployment.

## ⚙️ How to Run

Every project in this repository has been structured as a standalone, production-ready Python application. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/WajidAyub/Deep-Learning-Portfolio.git
   cd Deep-Learning-Portfolio
   ```
2. **Explore a project:**
   Navigate into any of the 6 project folders and read its dedicated `README.md` for specific execution instructions. Most scripts will generate their own performance evaluation graphics seamlessly.
